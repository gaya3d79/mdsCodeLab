# Tableau Dashboards

## Dashboard Overview

This project includes interactive Tableau dashboards developed to support data driven decisionmaking in University Admissions and Enrollment Management.
The dashboards complement the pmodelling results by providing visual insights for admissions, enrollment, institutional characteristics and yield patterns.

---

## Dashboard 1: Admissions & Enrollment Overview

### Objective
Provide an overview of institutional admissions and enrollment performance.

### Key Insights
1. Enrollment trends across institutions
2. Admission and acceptance patterns
3. Yield rate distribution
4. Institutional comparisons

### Dashboard Preview

![Dashboard 1](dashboard/gayathri.danappal_dashboard1.png)

---

## Dashboard 2: Predictive Modelling Insights

### Objective
Visualise the key metrics influencing enrollment outcomes and model performance.

### Key Insights
1. Feature importance analysis
2. Enrollment prediction results
3. Institutional segmentation
4 High yield and low yield institution identification
